# Smart Next Word Predictor

A Flask-based NLP project that predicts the next word using a Bigram (N-gram) language model.

## Features
- Real-time next word prediction
- Top 5 smart suggestions
- Emoji suggestions
- Interactive UI
- Built with Flask and NLTK

## Tech Stack
- Python
- Flask
- NLTK
- HTML
- CSS
- JavaScript
